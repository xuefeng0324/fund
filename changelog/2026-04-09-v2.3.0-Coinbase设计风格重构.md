# v2.3.0 - Coinbase 设计风格重构

## 概述
全面重构 UI 为 Coinbase 设计风格，统一颜色系统、圆角规范和组件样式。

## 变更内容

### 设计规范
- 新增 DESIGN.md 设计规范文档，定义 Coinbase 风格颜色、字体、圆角系统
- 统一颜色系统：
  - 品牌色：#0052ff (Coinbase Blue)
  - 深色：#0a0b0d (Near Black)
  - 背景：#ffffff (Pure White)
  - 次级背景：#eef0f3 (Cool Gray Surface)
  - 悬停色：#578bfa (Hover Blue)
  - 边框：rgba(91, 97, 110, 0.2)

### 圆角规范
- 按钮/输入框：56px (Pill 形态)
- 卡片/表格：16px
- 对话框：24px
- 标签：56px

### 组件样式
- 表格头部背景：#0052ff，悬停变浅蓝 #578bfa
- 标题蓝色装饰条：4px 宽，#0052ff
- 涨跌颜色：红色涨、绿色跌

### 指数卡片
- 四行显示：名称、价格、涨跌、百分比
- 宽度对齐基金表格内容区域
- PC端居中显示

### 管理基金弹窗
- 全端适配：PC端 480px / 移动端 96%
- 删除图标替换为 CircleCloseFilled
- 悬停红色背景效果
