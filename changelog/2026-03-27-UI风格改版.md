# 2026-03-27 UI风格改版与功能优化

## 变更内容

### 1. 页面风格改版
- 采用浅色撞色设计风格
- 主色调：靛蓝 `#4f46e5`（按钮、表头、高亮）
- 涨跌色：红 `#dc2626` / 绿 `#16a34a`
- 背景：浅灰白 `#fafafa`
- 卡片：白色圆角设计

### 2. 组件优化

#### Toolbar 工具栏
- 添加「看自己 | 看全部」切换按钮组
- 输入框加长（搜索 160px，密钥 100px）
- 按钮统一圆角 8px，靛蓝色调

#### FundRow 基金行
- 全表粗体显示（font-weight: 600）
- 统一字体大小 14px
- 建议/原因保持持仓/空仓多行显示

#### IndexStrip 指数条
- 改为白色卡片风格
- 字体加粗、颜色加深更易读

#### FundManageModal 管理基金弹窗
- 全新卡片式布局
- 基金列表独立卡片，带悬停效果
- 添加基金实时显示名称
- 圆角按钮与整体风格统一
- **新增**：支持通过 GitHub API 保存基金配置

### 3. GitHub API 集成
- 添加 `src/api/github.js` 封装 GitHub Contents API
- 支持读取和更新 `fund_codes.json`、`fund_groups.json`
- 配置 `VITE_GITHUB_TOKEN` 环境变量
- 指定 `lyl-dev-claude` 分支进行读写操作

### 4. 生产环境修复
- 修复 `pingzhongdata` 接口在生产环境 404 问题
- 生产环境直接访问 `fund.eastmoney.com` 源站
- 开发环境继续使用 Vite 代理

### 5. Bug 修复
- 修复「看全部」清空密钥输入框的问题
- 修复管理基金列表不显示基金名称的问题
- 修复点击切换视图时密钥状态不同步的问题

- `src/assets/style.css` - 全局样式改版

- `src/components/FundRow.vue` - 表格行样式

- `src/components/Header.vue` - 头部简化
- `src/api/github.js` - 新增 GitHub API 封装
- `src/api/funds.js` - 修复生产环境 API 调用
