# v2.4.3 (2026-04-14) - 移除 FundMNFInfo 接口

## 变更信息

**API 移除与简化**
- 彻底移除了对 `fundmobapi.eastmoney.com/FundMNewApi/FundMNFInfo` 接口的调用。
- 移除了关联函数，如 `fetchRealtimeBatch` 和 `fetchFundBasicInfo`。
- 将 `useFunds.js` 中的批量估值获取逻辑变更为纯粹依赖单只基金补齐（`fundgz.1234567` 接口）和涨跌信息（`pingzhongdata` 接口）。

**变更原因**
- 在纯前端（浏览器）环境下，`FundMNFInfo` 经常因严格的 User-Agent 校验和跨域限制（CORS）报错。
- 为了保证 GitHub Pages 和本地开发环境的一致性和稳定性，改为完全依赖支持跨域读取的 JSONP 方案。
