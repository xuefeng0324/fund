# v2.1.11 (2026-04-02)

## UI 优化

### 加载状态优化
- 更新中禁用看自己/看全部、排序、展开全部按钮
- Toolbar 使用 isLoading 组合状态（loading || adviceLoading）
- FundTable 移动端排序和展开按钮使用 isLoading

### 密钥清空修复
- 清空密钥后自动切换看全部并刷新数据
- App.vue watch(keyValue) 处理清空时清除 validKey 和 localStorage
- Toolbar.vue 添加 @clear 事件处理密钥清空按钮

### 无密钥禁用开关
- 无有效密钥时禁用看自己/看全部开关
- handleSwitchChange 和 handleSwitchClick 检查 validKey

### 移动端更新时间显示
- PC 端隐藏第三行更新时间（仅移动端显示）
- 修复 CSS 选择器，移动端正确显示第三行

### 管理基金列表优化
- 移动端基金名称显示宽度调整为 180px
- PC 端完整显示，移动端截断

## 配置刷新

### 保存后立即生效
- FundManageModal: save 成功后 emit 更新的数据
- App.vue: onFundSaved 直接更新本地状态再 loadData
- 避免 GitHub Pages 部署延迟导致配置不刷新

## 请求优化

### 间隔调整
- funds.js: fundgz 请求队列间隔 100ms → 150ms
- useAdvice.js: pingzhongdata 请求间隔 100ms → 150ms

## 开发环境

### 移除代理配置
- vite.config.js: 移除开发环境代理
- funds.js: pingzhongdata 直接请求外部 API
- 统一开发环境与生产环境请求方式
