# 2026-04-02 - 右上角时间与移动端指数优化

## 概述

本次更新优化了 Header 右上角时间显示，并改进了移动端指数卡片的布局。

## 变更详情

### Header.vue
- 移除数据更新时间显示，改为显示当前时间
- 时间格式：`yyyy-MM-dd HH:mm:ss`，每秒更新
- 字体加粗显示
- 桌面端和移动端都显示当前时间

### IndexStrip.vue
- 移动端指数卡片改为每行 4 个显示
- 4 行布局：指数名称（加粗）、净值（加粗）、涨跌净值、涨跌幅（加粗）
- 添加 min-width: 0 解决 flex 换行问题
- 优化样式确保在小分辨率下正常显示

### App.vue
- 移除不再使用的 lastUpdate prop 传递

## 测试

- [x] 桌面端右上角时间显示正常
- [x] 移动端指数一行 4 个显示
- [x] 分辨率 357x667 显示正常
