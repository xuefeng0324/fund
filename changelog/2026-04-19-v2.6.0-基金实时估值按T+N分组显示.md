# v2.6.0 (2026-04-19)

**功能优化 — 基金实时估值按买入确认日分组显示**

将基金实时估值数据按买入确认日（T+N）分组显示，每个分组对应一个可折叠的 FundTable 表格。

## 主要变更

- 新增 `fundInfoMap` 从 `fund_info.json` 导入基金交易规则数据
- FundTable 组件新增折叠功能（`collapsible` 和 `defaultCollapsed` props）
- App.vue 按 `buy_confirm_date` 分组渲染多个 FundTable

## 功能说明

- **分组规则**：按 `buy_confirm_date` 值分组（1=T+1, 2=T+2, ...）
- **无记录基金**：归为"未知"组，显示在最后
- **折叠功能**：表格标题可点击折叠/展开
- **刷新状态**：刷新后所有表格默认展开（无需持久化）

## 技术实现

- `useConfig.js` 导入 `fund_info.json` 并构建 `fundInfoMap`
- `FundTable.vue` 支持 `collapsible` 和 `defaultCollapsed` props
- `App.vue` 使用 `groupedFunds` computed 按 T+N 分组
