# v2.6.1 (2026-04-19)

## 变更类型

UI 优化

## 变更内容

### 折叠箭头 SVG 图标

将 FundTable 组件的折叠箭头从 Unicode 字符（▶/▼）改为 SVG chevron 图标，符合 Coinbase 设计风格。

**变更文件：** `src/components/FundTable.vue`

### 具体改动

1. **模板部分**
   - 用 SVG chevron 替代 Unicode 字符

2. **样式部分**
   - `.collapse-icon`: 添加 `inline-flex` 居中，hover 时变为 Coinbase Blue
   - `.chevron`: 14×14px SVG 图标，添加旋转动画（收起时旋转 -90°）
   - 悬停时颜色从 `#5b616e` 变为 `#0052ff`（Coinbase Blue）

### 设计规范依据

参考 `DESIGN.md` 中的 Coinbase 设计风格：
- 品牌色：#0052ff 用于交互状态
- 图标尺寸：14-16px 范围
- 过渡动画：0.2-0.25s ease

### 视觉效果

- **默认状态**: chevron 向下（展开），颜色 #5b616e
- **折叠状态**: chevron 旋转 -90°（向左），颜色 #5b616e
- **hover 状态**: 颜色变为 #0052ff（Coinbase Blue）
- **动画**: transform 0.25s ease 平滑旋转
