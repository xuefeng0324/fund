# 净值已更新功能

## 需求背景

根据基金买入确认日（T+N）的不同，净值更新时间也不同：
- T+1：今天（工作日）更新
- T+2：昨天（今天-1工作日）更新
- 以此类推...

需要根据历史净值日期判断基金是否已更新，已更新时在基金名称后显示徽标。

## 实现方案

### 1. 逻辑判断（useFunds.js）

```javascript
// T+N 规则：期望历史日期 = 今天 - (N-1) 个工作日
// T+1: today - 0 = 今天
// T+2: today - 1 = 昨天
```

### 2. UI 显示（FundTable.vue）

- 基金名称后添加蓝色药丸徽标"已更新"
- 已更新时净值、涨跌幅取历史数据
- 徽标样式：透明背景 + #0052ff 边框 + 100000px 圆角
