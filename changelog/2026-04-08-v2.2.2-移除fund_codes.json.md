# v2.2.2 - 移除 fund_codes.json 配置文件

**发布日期**: 2026-04-08

## 变更说明

移除 `fund_codes.json` 配置文件，基金代码由 `fund_groups.json` 中所有分组的代码合并后去重生成。

## 技术变更

### 文件变更

| 文件 | 操作 |
|------|------|
| `public/config/fund_codes.json` | 删除 |
| `src/composables/useConfig.js` | 修改 |
| `src/components/FundManageModal.vue` | 修改 |

### useConfig.js 变更

- `fundCodes` 改为 computed 属性，由 `fundGroups` 中所有分组的代码合并去重生成
- `loadConfig()` 只加载 `fund_groups.json`，不再加载 `fund_codes.json`
- `loadConfigFromGitHub()` 只加载 `fund_groups.json`
- 缓存逻辑简化，只缓存 `fundGroups`

### FundManageModal.vue 变更

- `loadAllConfig()` 只加载 `fund_groups.json`
- `save()` 只保存 `fund_groups.json`，不再同步 `fund_codes.json`
- 移除 `codesSha`、`allCodes` 变量

## 影响范围

- 配置管理逻辑简化，只需维护一份 `fund_groups.json` 配置文件
- 基金代码列表自动由各分组代码合并去重生成
- 管理基金列表保存时，只需更新分组配置
