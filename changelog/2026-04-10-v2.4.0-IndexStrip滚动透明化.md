# v2.4.0 - IndexStrip 滚动透明化效果

**发布日期**: 2026-04-10

## 新功能

### IndexStrip 滚动透明化效果

页面下滑时指数数据区域逐渐透明，直到"实时估值基金"表格的顶部接近 sticky header 时完全透明。向上滚动时效果相反。

## 技术实现

### 核心逻辑

- **requestAnimationFrame 节流**: 使用 RAF 包装滚动回调，防止同一帧内多次触发计算
- **passive scroll listener**: 提升滚动性能
- **动态 fade distance**: 基于 IndexStrip 高度动态计算淡出起始距离（stripHeight + 50px）
- **ease-out cubic 缓动**: 开始变化快、结束变化慢，过渡更自然
- **阈值检查**: 只在 opacity 变化超过 0.01 时触发响应式更新，减少不必要的渲染

### 关键函数

```javascript
// 缓动函数
function easeOutCubic(t) {
  return 1 - Math.pow(1 - t, 3)
}

// 动态计算淡出距离
function getFadeDistance() {
  const indexStripEl = stickyHeaderEl.value?.querySelector('.index-strip')
  if (!indexStripEl) return 200
  return indexStripEl.offsetHeight + 50
}

// 计算 opacity
function calculateOpacity() {
  const distance = tableTop - headerBottom
  const progress = Math.min(Math.max(distance / fadeDistance, 0), 1)
  return easeOutCubic(progress)
}
```
